E-plantShopping App
A modern, responsive e-commerce web application designed for plant lovers to discover, browse, and purchase indoor and outdoor plants.

Features:
Plant Catalog - Filter plants by category, sunlight needs, and price.Smart Search - Find specific plants with predictive search results.
Shopping Cart - Dynamic cart management with instant price updates.
Secure Checkout - Integrated mockup payment system for seamless orders.User Accounts - Order history tracking and personalized plant wishlists.
